Cursor 于 2023 年推出后迅速成为 AI 编程领域的标杆产品。它不是简单地在编辑器里加个聊天窗口，而是把 AI 能力深度嵌入到编辑器的每一个角落。

## 适合哪些人用

- **初学者**：AI 解释报错、自动补全，学习曲线大幅降低
- **全栈开发者**：跨文件理解项目结构，Composer 模式一次改多个文件
- **独立开发者**：一个人当多人用，快速实现功能原型

## 和 GitHub Copilot 的区别

Cursor 的 Agent/Composer 模式远比 Copilot 强大——Copilot 主要做行级补全，而 Cursor 能理解整个项目并自主完成任务。但 Copilot 与 GitHub 生态集成更好，企业用户更常选择 Copilot。
