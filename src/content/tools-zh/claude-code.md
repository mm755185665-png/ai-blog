Claude Code 是 Anthropic 于 2025 年推出的终端原生 AI 编程工具，基于 Claude Sonnet 和 Opus 系列模型构建。与 GitHub Copilot 等行内补全工具不同，Claude Code 定位为 Agentic 编程助手——它不只是提示下一行代码，而是能够理解整个项目结构，自主完成"修复 Bug → 运行测试 → 提交代码"的完整工作流。
