GitHub Copilot 于 2021 年由 GitHub 与 OpenAI 联合推出，是目前市场占有率最高的 AI 编程助手之一。它基于 OpenAI Codex 模型，能够根据注释、函数签名和上下文自动生成代码片段，大幅降低重复性编码工作量。
